# Order Processor

Starter files for the Code Review and Refinement lab. The code runs and the minimal test suite passes — that does not mean it is correct. Your job is to review this code with GitHub Copilot and decide what should change.

See `code-review-and-refinement.adoc` for the full lab guide (setup steps, review loop, and evaluation criteria). A reference implementation of the reviewed-and-fixed repository lives under `solutions/`.
